🍏 iOS / macOS

> Рекомендуемый клиент: Streisand

Скачать для iOS / macOS
1. Установите Streisand из App Store.
2. В боте @givpn_bot нажмите «Личный кабинет» → скопируйте ссылку подписки (рекомендуется) или VLESS-ссылку.
💡 Ссылка подписки автоматически обновляет конфигурацию, вам не придётся менять ключи вручную.
3. Откройте Streisand, нажмите на значок «+» в правом верхнем углу.
4. Выберите «Импортировать из буфера обмена».
5. Нажмите на кнопку подключения (переключатель). При первом запуске разрешите добавление VPN‑профиля.
✅ Вы защищены! Ваш трафик зашифрован и анонимен.

📱 Android / Windows

> Рекомендуемый клиент: v2rayNG

Скачать для Android Скачать для Windows
1. Установите v2rayNG из магазина или с GitHub.
2. В боте @givpn_bot перейдите в «Личный кабинет» → скопируйте ссылку подписки (рекомендуется).
💡 Ссылка подписки позволяет автоматически получать актуальные ключи.
3. Откройте v2rayNG, нажмите на значок «+» (или «≡» → «Импорт»).
4. Выберите «Импорт из буфера обмена».
5. Нажмите на кнопку подключения (иконка ракеты/зонтика).
✅ Вы защищены! Добро пожаловать в свободный интернет.
